# ping-pong-coop-game
Игра пинг-понг написана на языке Python и использует библиотеку PyGame

Движение ракетки слева реализуется через клавиши W и S, движение ракетки справа реализуется черз стрелочки "Вверх" и "Вниз"

Использовать данный код вы можете только при указании моего авторства.
